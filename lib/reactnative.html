# 解决 expo 连不上你真实设备的问题

You can get around that by fixing your laptop's ip address to a static ip, let's say 192.168.1.69, then you need to change the react native packager ip address to the same of your laptop, usingset REACT_NATIVE_PACKAGER_HOSTNAME=192.168.1.69 (Windows) REACT_NATIVE_PACKAGER_HOSTNAME=192.168.1.69 (Mac & Linux) start your project again and you're good to go.
