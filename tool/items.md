# 命令行 iterm

iterms 比较好用的一点，在命令行输入 d ，就可以输出 10 个你使用的目录，然后通过数字来跳转，真的非常方便

## 几个快捷键

- Ctrl + A：移动到行首
- Ctrl + E：移动到行尾
- Ctrl + K：删除到行尾

## mac 使用自定义快捷键调出终端

推荐使用 iTerm2，比自带的强大一些。

`iTerm2 -> Preferences -> Appearance -> Hotkey -> Check Show/hide`
