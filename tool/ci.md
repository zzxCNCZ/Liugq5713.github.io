# 持续构建 Continuous integration

## 什么是 CI

持续集成指的是，频繁地（一天多次）将代码集成到主干

## CI 是如何工作的

1. 开发者编写代码，提交 commit 到仓库
2. CI 服务检测到更改
3. CI 编译系统，
4.
